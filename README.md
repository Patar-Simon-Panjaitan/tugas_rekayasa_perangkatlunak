# tugas_rekayasa_perangkat_lunak

Coding Standards yang Diterapkan
- Menggunakan nama variabel yang jelas (scores, average, student).
- Menggunakan Class untuk merepresentasikan objek mahasiswa.
- Menggunakan fungsi terpisah (input_scores(), main()).
- Menggunakan docstring pada class dan method.
- Menggunakan indentasi 4 spasi sesuai standar PEP 8.
- Menghindari duplikasi kode.
